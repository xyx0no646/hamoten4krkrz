# hamoten4krkrz

吉里吉里2で有名なテンプレートである「鱧天」を吉里吉里Zで動くようにしたものです。
最終的に吉里吉里SDL2経由でWeb上で動くようにしたい。

## ライセンス

お読みください.txt参照

## 使い方

tvpwin32.exeをダブルクリックします。

## IMPORTANT

当たり前のようにまともに動きません。
一部のメニューを選択するとアプリがクラッシュします。
